pip 配置

Posted on 2018-02-12 | In 未分类

镜像源:

http://mirrors.aliyun.com/pypi/simple/
http://pypi.douban.com/simple/
https://pypi.tuna.tsinghua.edu.cn/simple/
http://pypi.mirrors.ustc.edu.cn/simple/
https://pypi.mirrors.ustc.edu.cn/simple/

设置方法（没有对应的文件夹以及文件直接创建即可）:

  • linux: ~/.pip/pip.conf

  • windows: C:\Users\**\pip\pip.ini

内容如下（根据选择的镜像的传输协议(http或https）选择下面对应的内容填充）:

  • http源设置:

[global]
index-url = http://pypi.mirrors.ustc.edu.cn/simple/
trusted-host = pypi.mirrors.ustc.edu.cn

  • https源设置:

[global]
index-url = https://pypi.mirrors.ustc.edu.cn/simple

docker 安装mysql及其简单的配置记录

Posted on 2018-02-05 | In 未分类

拉取mysql 5.7 镜像

docker pull mysql:5.7

创建实例，并设置数据持久化以及配置mysql数据库支持中文并且设置数据库默认引擎为MyISAM（设置字符集为utf8):

说明：
设置数据持久化: -v E:\\databases\mysql_data:/var/lib/mysql
设置中文支持: –character-set-server=utf8mb4 –collation-server=utf8mb4_unicode_ci
设置默认数据引擎: –default-storage-engine=MyISAM
